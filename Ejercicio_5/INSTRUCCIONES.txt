INSTRUCCIONES:

1- CORRER EL SERVIDOR PRIMERO ANTES DE CORRER LOS CLIENTES. ENTRAR A LA CARPETA /bin/ ANTES DE EJECUTAR
	EL EJECUTABLE DEL SERVIDOR ESTA EN servidor/bin/servidor
	EL EJECUTABLE DEL CLIENTE ESTA EN cliente/bin/cliente

2- CERRAR EL SERVIDOR SIEMPRE ULTIMO.

3- EL SERVIDOR SE CIERRA CON SIGINT O SIGTERM, RECOMENDADO USAR EL CTRL+C

4- NO CORRER MULTIPLES SERVIDORES.

5- ARCHIVOS DE CONFIGURACION EN cliente/conf/config.conf servidor/conf/config.conf

6- CUALQUIER CONFIGURACION SERA APLICADA AL REINICIAR EL SERVIDOR/CLIENTE SI ES QUE YA ESTABAN CORRIENDO

7- LA BASE DE DATOS ESTA EN servidor/db/db.dat. SI SE SUMINISTRA SU PROPIA BASE DE DATOS QUE EL FORMATO RESPETE EL EXISTENTE (ESPECIFICADO EN LA CONSIGNA DEL EJERCICIO) O NO SE GARANTIZA FUNCIONAMIENTO

8- ACTIVAR MODO DEBUG EN LA CONFIGURACION PARA MAS INFORMACION EN PANTALLA SOBRE LAS OPERACIONES

9- ACTIVAR AYUDA EN LA CONFIGURACION PARA MAS INFORMACION.
